# CalorieTracker
Privacy Policy — Calorie Tracker (Local)

This app stores food entries and calorie values locally on your device using an on-device SQLite database.
The app does not require an account or login.

Data collection:
- The developer does not collect any personal information.
- The app does not transmit user data to external servers.

Data storage:
- Entries are stored locally within the app’s private storage on the device.
- Uninstalling the app removes locally stored data.
